# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal
attacks are not tolerated. Critique ideas and code, not people. Assume good faith and
favor evidence over assertion — the same standard the tool itself applies.

## Enforcement

Report unacceptable behavior to the maintainer listed in `KEYS.md` (or via GitHub
private contact). Reports are handled confidentially. Maintainers may remove, edit, or
reject contributions and comments that violate this policy, and may temporarily or
permanently ban contributors for serious or repeated violations.

The full text and enforcement guidelines are available at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
